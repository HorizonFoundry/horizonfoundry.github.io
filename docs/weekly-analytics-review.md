# Weekly Analytics Review

## Purpose
This review exists to keep Horizon Foundry aligned to measurable value.

The goal is not to chase vanity metrics.
The goal is to understand:
- what content is attracting attention
- what channels are sending the right visitors
- what calls to action are getting intent
- what topics are earning search visibility
- what should change in the next publishing cycle

## Cadence
Run this review once per week.

Recommended time:
- Monday morning, 15 to 20 minutes

## Sources
- GA4
- Google Search Console

## Review questions
Answer these each week:

1. Which pages attracted the most attention?
2. Where did visitors come from?
3. Which CTAs were clicked?
4. Which topics/pages are starting to show up in search?
5. What does this suggest I should publish, refine, or promote next?

## Weekly review steps

### 1. GA4: traffic and attention
Check:
- top pages by views
- landing pages
- traffic sources / referrers
- engaged sessions or engagement time, if useful

Write down:
- top 3 pages this week
- top 3 traffic sources this week
- any page that drew attention unexpectedly

### 2. GA4: CTA intent
Filter to:
- event name = `cta_click`

Break down by:
- `CTA Label`
- `CTA Location`
- `CTA Type`

Write down:
- total CTA clicks this week
- which CTA got the most clicks
- which page/location produced the most CTA activity
- any CTA that is being seen but not clicked much

### 3. Search Console: search visibility
Check:
- top pages by impressions
- top queries
- clicks
- CTR
- average position

Write down:
- top 5 queries
- top 5 pages by impressions
- pages with impressions but low CTR
- pages that may need a stronger title or meta description

### 4. Interpretation
Complete these prompts:

- The content theme getting the most traction right now is:
- The channel sending the most useful attention right now is:
- The CTA performing best right now is:
- The most obvious content opportunity is:
- The weakest point in the funnel right now is:

### 5. Action decision
Choose no more than 3 actions for the coming week.

Examples:
- publish a follow-on essay on a topic showing early traction
- improve internal links on a high-impression page
- sharpen a weak CTA label or placement
- share a strong page again on LinkedIn
- revise title/meta description for a page with impressions but weak CTR

## Guardrails
- Do not optimize for raw pageviews alone.
- Do not make content decisions from one noisy week.
- Do not create more metrics than can actually inform action.
- Prefer signal that supports authority, engagement quality, and publishing discipline.

## Monthly checkpoint
At the end of each month, review the last four weekly entries and answer:

- Which themes are compounding?
- Which channels are consistently useful?
- Which CTAs actually produce intent?
- Which essays deserve follow-up, expansion, or an artifact?
- What should be deprioritized?
